## Standards techniques imposés

### Codage

- **Type-safety**: Strict typing, Zod pour validation, zéro compromise sur la sécurité des types
- **Données monétaires**: Entiers (centimes) uniquement. Decimal.js pour logique comptable
- **Documentation**: Living Doc approach, TypeDoc API + Storybook composants
- **IA & Privacy**: Privacy-first, connecteurs interchangeables (Ollama local préféré)

### Infrastructure

- **Hosting**: Self-hosted VPS durci (Alpine/Debian)
- **CI/CD**: Dual-remote (GitHub Public + GitLab Private/Sovereign)
- **Monitoring**: Netdata/Glances (low overhead) + Uptime Kuma alertes critiques
- **Backup 3-2-1**: Restic/BorgBackup + S3 Souverain (Scaleway/OVHcloud/MinIO)

### Compliance

- **Fiscalité**: Factur-X native, PPF/PDP 2026-ready
- **Privacy**: Privacy-by-design, IndexedDB/SQLite, zéro télémétrie sans consentement
